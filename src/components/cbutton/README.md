### how to install ?
`npm install react-native-common_button --save`
or use yarn
`yarn add react-native-common_button`

### how to use
```
import Btn from 'react-native-common_button'
 _onBtnB() {
        alert('Btn click in');
    };
<Btn onBtnClick={this._onBtnB}/>
```

more props are follows:
```
btnText: string,
btnColor: object,
btnWidth: number,
btnHeight: number,
btnRadius: number,
btnTextSize: number,
btnTextColor: object,
onBtnClick: func.isRequired
```
